# HASPEN Web Sitesi

Zonguldak Kilimli'de faaliyet gösteren HASPEN firması için modern, responsive web sitesi.

## 🎯 Özellikler

- ✅ Tek sayfa (Single Page) modern tasarım
- ✅ Mobil uyumlu (Responsive)
- ✅ WhatsApp entegrasyonu
- ✅ Teklif formu
- ✅ Google Maps entegrasyonu
- ✅ Smooth scroll animasyonları
- ✅ SEO dostu yapı

## 📦 İçerik

- **Ana Sayfa (Hero)**: Etkileyici giriş alanı
- **Hakkımızda**: Firma bilgileri, vizyon, misyon
- **Ürünler**: PVC, Alüminyum, Çelik Kapı, Cam Balkon, Duşa Kabin
- **Galeri**: Proje görselleri (placeholder)
- **Teklif Al**: İletişim formu
- **İletişim**: Telefon, adres, harita

## 🚀 Kurulum

1. Dosyaları web sunucunuza yükleyin
2. `images/` klasörüne proje görsellerini ekleyin
3. Google Maps koordinatlarını güncelleyin
4. Sosyal medya linklerini ekleyin

## 📝 Yapılacaklar

- [ ] Gerçek proje fotoğrafları ekle
- [ ] Logo tasarımı ve ekleme
- [ ] E-posta adresi güncelle
- [ ] Instagram ve Facebook linkleri ekle
