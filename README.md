# HASPEN Web Sitesi

Zonguldak Kilimli'de faaliyet gösteren HASPEN firması için ultra profesyonel, minimal web sitesi.

## 🎯 Özellikler

- ✅ Profesyonel minimal tasarım (Beyaz-Siyah-Gri)
- ✅ Tek sayfa (Single Page) modern yapı
- ✅ Tam responsive (Mobil uyumlu)
- ✅ WhatsApp entegrasyonu
- ✅ Üretim süreci timeline
- ✅ Proje galerisi (filtrelenebilir)
- ✅ SSS (Accordion)
- ✅ Blog bölümü
- ✅ Sertifikalar & Markalar
- ✅ Smooth scroll animasyonları
- ✅ SEO dostu yapı
- ✅ Google Maps entegrasyonu

## 📦 İçerik Bölümleri

1. **Hero Section**: İstatistikler ve CTA butonları
2. **Hakkımızda**: Firma bilgileri, özellikler
3. **Ürünler**: 5 kategori (PVC, Alüminyum, Çelik Kapı, Cam Balkon, Duşa Kabin)
4. **Üretim Süreci**: 5 adımlı timeline
5. **Projeler**: Filtrelenebilir galeri
6. **Referanslar**: Müşteri yorumları
7. **SSS**: 8 soru-cevap
8. **Sertifikalar**: Çalışılan markalar
9. **Blog**: 3 makale
10. **Teklif Formu**: WhatsApp entegrasyonlu
11. **İletişim**: Harita + bilgiler

## 🚀 Kurulum

1. Dosyaları web sunucunuza yükleyin
2. `images/` klasörüne görselleri ekleyin
3. Google Maps koordinatlarını güncelleyin
4. Sosyal medya linklerini aktif edin
